Требования
ОС: Windows
КОМПАС-3D: с поддержкой API7
.NET Framework: 4.8
Используемые технологии и библиотеки
C# / .NET Framework 4.8
KompasAPI7 — работа с документами КОМПАС
COM Interop — взаимодействие с API КОМПАС
NPOI — экспорт и импорт Excel# KompasDropExport

*KompasDropExport*

Инструмент для работы с проектами КОМПАС-3D:
анализ связей, массовое переименование и обновление ссылок, экспорт файлов.

*Возможности*
Анализ проекта
поиск всех связанных файлов
построение дерева зависимостей (A3D, M3D, CDW, SPW)
Массовое переименование
изменение обозначения и наименования
табличный режим редактирования
экспорт / импорт через Excel
Обновление ссылок

Автоматически обновляются связи после переименования:
CDW → 3D модель
SPW → вложенные документы
SPW → документы внутри спецификации
(SpecificationDescriptions → BaseObjects → AttachedDocuments)
A3D/M3D → attached documents
A3D → состав сборки

Экспорт:
экспорт моделей в STEP
экспорт чертежей в PDF

Как работает:
Сканирование папки проекта
Построение графа связей
Формирование списка переименований
Применение rename
Автоматическое обновление ссылок
Сохранение документов


Ограничения:
используется COM API КОМПАС (нестабильное поведение возможно)
часть ссылок может не обновляться (зависит от типа документа)
корректность fallback по имени файла зависит от уникальности имён

Структура проекта:
Domain   — модели данных  
Services — логика (анализ, rename, экспорт)  
Kompas   — работа с API7  
UI       — интерфейс  
Utils    — вспомогательные функции 

Ускоряет работу с большими проектами КОМПАС:

устраняет ручное обновление ссылок
снижает риск ошибок при переименовании
упрощает экспорт и подготовку данных
